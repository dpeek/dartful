Dart Markdown

 Friday, 22 February 2013

Minor detour. My questions on the dart mailing list have lead to my first library on pub.dartlang.org. While I can't take credit for the code (I think Bob Nystrom wrote most of it), I'm happy to make myself useful by maintaining and building on it.

Extracting the library from Dartdoc meant removing support for Dart syntax highlighting. Fortunately, with some pointers from the list, I've written a new Dart classifier based on analyzer_experimental. HTML/CSS classifiers are also in the works.

The project is located here, and there is a editor demo running on GitHub pages. Pull requests, feedback and issues welcome.
