Dartful

A Simple Dart Blogging Engine

Friday, 15 February 2013

This blog documents my experience learning Dart, getting to know the language, libraries, toolchain, and community. As a learning project I'll be building a simple static site generator similar to Jekyll. I don't have big plans for the project, but hopefully others will find it a useful reference.

The source for this site is available on GitHub.

Objectives

The goal is to convert a collection of Markdown documents into HTML: an index and a page for each post. Additional post metadata will be defined in a block of YAML before the content, parsed into a model and used to generate each page.

Here are the requirements:

  • Listing and reading markdown from the file system
  • Parsing the YAML metadata
  • Converting Markdown to HTML, hopefully with support for syntax highlighting
  • Generating HTML for the index and pages from templates
  • Paginating the index (once I have enough posts)
  • Generating an RSS feed
